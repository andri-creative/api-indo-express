GET /api/v1/regencies?province_code=11

Mendapatkan kabupaten/kota berdasarkan kode provinsi 

{
  "success": true,
  "data": [

    },
    ...
  ]
}
GET /api/v1/districts?regency_code=11.01

Mendapatkan kecamatan berdasarkan kode kabupaten/kota

GET /api/v1/villages?district_code=11.01.01

Mendapatkan kelurahan/desa berdasarkan kode kecamatan

Tentang API Indonesia

API Indonesia adalah layanan gratis yang menyediakan data wilayah administratif Indonesia secara lengkap dan terstruktur. Dibangun dengan teknologi modern seperti Express.js dan TypeScript untuk performa dan keamanan optimal.
